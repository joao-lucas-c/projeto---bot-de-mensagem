Protótipo de Automação( Bot de Mensagens WhatsApp ):

- Este é um protótipo funcional de um bot de WhatsApp desenvolvido em Python (pywhatkit).

- O projeto marca meu início na área de automação, focando no envio simplificado de mensagens 
enquanto aprimoro o uso de bibliotecas como OpenPyXL (Excel) e Flask. 
É uma base de estudos em constante evolução.
